# chapter5_practice

A Flutter food category grid practice app for Chapter 5. This project focuses on GridView layouts, reusable widgets, and navigation to detail screens.

## Features

- Food category grid layout
- Reusable category grid item widget
- Detail screen for selected categories
- Local image assets for cuisine categories

## Main Files

- `lib/main.dart`
- `lib/screens/menu_page.dart`
- `lib/screens/details_page.dart`
- `lib/widgets/category_grid_item.dart`
- `lib/models/category.dart`

## How to Run

```bash
flutter pub get
flutter run
```
