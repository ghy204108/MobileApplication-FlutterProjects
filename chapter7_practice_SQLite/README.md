# chapter7_practice_SQLite

A Flutter SQLite CRUD notes app for Chapter 7. This project focuses on local database storage and basic create, read, update, and delete operations.

## Features

- Local SQLite database integration
- Notes list screen
- Create, update, and delete note records
- Database service layer using `sqflite`

## Main Files

- `lib/main.dart`
- `lib/homepage.dart`
- `lib/db_service.dart`

## How to Run

```bash
flutter pub get
flutter run
```
