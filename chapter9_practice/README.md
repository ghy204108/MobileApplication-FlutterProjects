# chapter9_practice

A Flutter OpenStreetMap practice app for Chapter 9. This project focuses on map rendering, markers, coordinates, and map interaction using `flutter_map`.

## Features

- OpenStreetMap display
- Map markers for selected locations
- Coordinate handling with `latlong2`
- Map zoom and interaction practice

## Dependencies

- `flutter_map`
- `latlong2`

## Main Files

- `lib/main.dart`
- `lib/homepage.dart`

## How to Run

```bash
flutter pub get
flutter run
```
