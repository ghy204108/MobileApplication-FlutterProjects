# chapter4_practice

A Flutter contact list practice app for Chapter 4. This project focuses on displaying structured data in a ListView and navigating to a details page.

## Features

- Contact list UI
- Contact model class
- ListView-based layout
- Navigation from a list item to a detail screen

## Main Files

- `lib/main.dart`
- `lib/homepage.dart`
- `lib/detailspage.dart`
- `lib/models/contact.dart`

## How to Run

```bash
flutter pub get
flutter run
```
