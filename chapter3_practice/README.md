# chapter3_practice

A Flutter restaurant and diet browsing UI practice project. It introduces structured UI screens, model classes, image assets, and SVG usage.

## Features

- Restaurant-style home page UI
- Category and diet model classes
- Asset-based UI content
- SVG support through `flutter_svg`

## Main Files

- `lib/main.dart`
- `lib/homepage.dart`
- `lib/mainrestaurant.dart`
- `lib/models/category.dart`
- `lib/models/diet.dart`

## How to Run

```bash
flutter pub get
flutter run
```
