# chapter6_practice

A Flutter BMI calculator practice app for Chapter 6. This project focuses on user input, TextField handling, basic calculations, and displaying results in the UI.

## Features

- Height and weight input fields
- BMI calculation logic
- Result display in a Flutter screen
- Basic form-style UI practice

## Main Files

- `lib/main.dart`
- `lib/homepage.dart`

## How to Run

```bash
flutter pub get
flutter run
```
