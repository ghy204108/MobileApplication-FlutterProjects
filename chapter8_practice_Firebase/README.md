# chapter8_practice_Firebase

A Flutter Firebase notes app for Chapter 8. This project focuses on connecting a Flutter app to Firebase and storing notes in Cloud Firestore.

## Features

- Firebase initialization
- Cloud Firestore integration
- Notes data stored in Firebase
- Basic Firebase-backed Flutter app structure

## Dependencies

- `firebase_core`
- `cloud_firestore`

## Firebase Setup

Create your own Firebase project and configure the required platform files before running the app. Local Firebase configuration files are not intended to be committed to this public repository.

## How to Run

```bash
flutter pub get
flutter run
```
